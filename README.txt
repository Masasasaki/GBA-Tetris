ScaryTetris by Masaki Asanuma

The classic game of Tetris with the falling speed of the block is randomized!
You level up every 10 points you score and the range of the randomized speed
will get smaller and smaller. For every tetris you get to see an image of
a cool dog!

Controls:

Start       Starts the game
Select      Resets the game to the title screen
A           Rotate left
B           Rotate Right
L           <None>
R           <None>
